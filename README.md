**Tickly** is a basic timer written in C. When you run it, it first displays the text `Tickly` in ASCII art, then lets you set minutes and seconds, and plays a beep sound when the time is up.

<img src="PREVIEW.PNG" alt="Tickly Preview" />

### How It Works
1. Run the program.  
2. The ASCII art `Tickly` is displayed.  
3. Enter the minutes and seconds for your countdown.  
4. When the timer ends, a beep sound alerts you.

### Features
- Shows `Tickly` in ASCII art at startup  
- Counts down minutes and seconds  
- Plays a beep sound when the timer ends  
- Simple and lightweight
